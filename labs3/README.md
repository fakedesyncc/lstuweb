# Лабораторная работа 3: Работа с базами данных (MySQL, PDO)

## Задание
1. Создать таблицу для хранения данных
2. Подключиться к базе через PDO
3. Перенести данные из CSV в MySQL
4. Вывести список записей в виде таблицы

## Настройка базы данных

Сначала создайте базу данных и таблицы:

```bash
# Подключитесь к MySQL контейнеру
docker exec -it autosalon_mysql mysql -uroot -proot

# Или выполните SQL файл
docker exec -i autosalon_mysql mysql -uroot -proot < database.sql
```

## Использование

Откройте в браузере: http://localhost:8080/labs3/index.php

## Функционал
- Подключение к MySQL через PDO
- Добавление автомобилей в базу данных
- Миграция данных из CSV (файл из labs2) в MySQL
- Отображение списка автомобилей из базы данных
