# Лабораторная работа 5: Авторизация и личный кабинет

## Задание
1. Создать таблицу пользователей
2. Реализовать регистрацию (password_hash())
3. Реализовать вход через сессии
4. Добавить роли пользователей

## Настройка базы данных

Выполните SQL для создания таблицы пользователей:

```bash
docker exec -i autosalon_mysql mysql -uroot -proot < database.sql
```

## Использование

Откройте в браузере: http://localhost:8080/labs5/index.php

## Функционал
- Регистрация пользователей с хешированием паролей (password_hash)
- Авторизация через сессии
- Система ролей (user, admin)
- Личный кабинет с информацией о пользователе
- Отображение автомобилей пользователя

## Тестовые данные

После регистрации вы можете войти в систему и просмотреть свой личный кабинет.
