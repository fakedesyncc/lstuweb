# Лабораторная работа 6: Генерация отчетов (PDF, Excel, CSV)

## Задание
1. Генерировать отчет (PDF, Excel, CSV) с данными из БД
2. Добавить кнопки скачивания

## Установка зависимостей

Убедитесь, что установлены необходимые библиотеки через Composer:

```bash
cd ../project
composer install
```

Или установите вручную:
- phpoffice/phpspreadsheet (для Excel)
- tecnickcom/tcpdf (для PDF)

## Использование

Откройте в браузере: http://localhost:8080/labs6/index.php

## Функционал
- Генерация отчетов в формате CSV
- Генерация отчетов в формате Excel (.xlsx)
- Генерация отчетов в формате PDF
- Кнопки для скачивания отчетов
- Отображение данных перед генерацией

## Форматы отчетов

1. **CSV** - простой текстовый формат с разделителями
2. **Excel** - формат .xlsx с поддержкой форматирования
3. **PDF** - документ с таблицей данных

Все отчеты содержат данные об автомобилях из базы данных.
