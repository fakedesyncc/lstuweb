# Изображения автомобилей

## Как добавить изображения

1. Поместите изображения автомобилей в эту папку (`/project/public/images/cars/`)
2. Используйте понятные имена файлов, например:
   - `vaz-2101.jpg`
   - `lada-granta.jpg`
   - `uaz-patriot.jpg`
   - `volga-21.jpg`

3. При добавлении автомобиля через форму укажите путь к изображению:
   - `/images/cars/vaz-2101.jpg`
   - `/images/cars/lada-granta.jpg`
   - и т.д.

## Рекомендуемые размеры

- Миниатюра в списке: 60x40px
- Полное изображение: до 800x600px
- Формат: JPG или PNG
