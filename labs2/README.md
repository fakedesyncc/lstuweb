# Лабораторная работа 2: Основы PHP и работа с формами

## Задание
1. Реализовать форму для ввода автомобиля
2. Обработать данные ($_POST, $_GET)
3. Сохранить введенные данные в CSV

## Использование

Запустите Docker из labs1:
```bash
cd ../labs1
docker-compose up -d
```

Откройте в браузере: http://localhost:8080/labs2/index.php

## Функционал
- Форма для добавления автомобиля (марка, модель, цена, год, цвет)
- Валидация обязательных полей
- Сохранение данных в CSV файл (cars.csv)
- Отображение списка добавленных автомобилей
